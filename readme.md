## Temper Assignment

- Clone the repository and run the following steps in order,
- ```cd temper_assignment```
- ``` composer install```
-  ```cp .env.example .env```
- ``` php artisan key:generate```.
- ``` php artisan migrate --seed```
- ``` php artisan serve```
- Visit ```http://127.0.0.1:8000/dashboard```
